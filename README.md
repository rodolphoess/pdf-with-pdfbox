# PDF with pdfbox

Simple project to test the library pdfbox for work with PDFs, whether reading, writing, or other possible operations.

O código para leitura é simples como na Main. O detalhe está na necessidade das libs commons-loggins-1.2, fontbox-2.0.19, pdfbox-2.0.19, pdfbox-tools-2.0.19, preflight-2.0.19, xmpbox-2.0.19
